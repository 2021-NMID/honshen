# 1. 文件的打开新建和储存

### 插入图片

1. 用文件中的打开选项：

缩放快捷键Alt+滚轮，左下角的像素为当前图片的尺寸，ppi表示当前图片的分辨率，旁边的小三角可以看当前文档的信息（比如大小等）

将两个图片放到一个项目中：文件->置入嵌入对象（先打开第一个图片，再置入其他的，需要多个项目，就文件->打开）

2. 直接双击工作区
3. 用拖动的方式，先找到图片，然后拖动

### 新建画布

文件->新建->创建

(一般显示器的尺寸为1920x1080，分辨率为决定图像精细程度的量级单位（像素/英寸），通常设置为72ppi,要打印的文档用300ppi,颜色模式通常用RGB和CMYK（需要打印出来的情况设置为CMYK）)

1）修改画布背景颜色：点击左边工具栏中的设置背景色，选择颜色->确定，Alt+backspace可以修改背景色

2）在图片中插入文字，按左边工具栏T键，按ctrl+t自由修改文字大小，或者按住ctrl+滚轮，ctrl+j可以在图层拷贝,然后可以用移动工具拖动出来，选中图层backspace可以删除

3）保存修改的效果：文件->存储为       (图片格式：PSD(PS的源文件格式，保存图层，方便二次修改)，JPG(常规图片保存格式)，PNG(可以存储透明背景效果，不会自动填充颜色)，TIFF(不经压缩，一般喷绘写真，印刷打印等需要))，在工作区空白处双击鼠标可打开图片

# 2.移动工具（图层概念）

### 图层概念

含有文字或图形的元素的胶片一张张从上到下顺序叠放在一起，按住Alt再点击图层旁的小眼睛处，可以单独显示一个图层，

### 移动工具

1)选中左边工具栏中的移动工具，在属性栏中设置好属性后（属性中的自动选择（可以按ctrl键）可以自动选择鼠标所在内容的图层，可以不用先选择图层，显示变换控件可以调节图层内容的大小和角度，默认等比例缩放，按住shift键可以自由调节大小，ctrl+t也可以调出自由变换），先选择需要移动的图层后，拖动图层上的内容即可。

2)按住ctrl可以选多个图层，然后新建一个组。拖动图层栏中的图层到另一个的下面，则另一个图层可以放入图层中，返回上一步可以用ctrl+z，一次性选择多个图层后，属性栏又可以调节几个图层的排列，其中还可以以画布或图层进行对齐,删除，新建图层等功能在图层栏的下边框

# 3.矩形选框工作组（选区概念）

### 选区概念和基本操作

就是对图像进行区域选择，即为选择后需要处理的部分

1）选取：图像->调整->色相饱和度，是针对所选图层操作的，若只有一个图层，要针对图层中的某一个地方进行操作，就可以用左边工具栏中的矩形选框工具（矩形选框：按住shift键可约束比例，画正方形，椭圆框可以画正圆，按住Alt可以从中心缩放圆，微调或变换可以用单击右键->变换选区，单行选区：选一行像素）

2）撤销：选取部分单击右键->取消选择（快捷键ctrl+D）

3）移动某个部分：选中后单击右键->通过拷贝的图层(或者用ctrl+j)来将此部分复制为一个图层，可以避免移动此部分时，原来的部分为空了

4)属性栏中的一些值：依次为新选区，加选区（增加选区，可以按住shift键），减选区（可以按住Alt键），选选区相交部分（按住shift+Alt键），羽化（可以让整个拷贝出来的边上有虚化的效果，数值大小n是将边框开始向内n像素内变虚化），样式，宽度，高度均是设置选区

5）填色：填充前景色(Alt+delet)，填充背景色（ctrl+delet）

# 4.套索工具组

1）套索工具：能够直接在原图上自由框选区域，属性与上述类似，要羽化某个选区可以在选择选区后单击右键选择羽化值

2）多边形套索工具：在两点间形成直线，形成多边形，按Alt键不松变成套索工具

3）磁性套索工具：选区和背景有色差的情况，先选中一个点开始，然后鼠标沿着要选中的区域移动，系统会根据与周围的色差自动生成结点，若色差不明显，可以手动点一个点，选好后按enter键选中，按back键可以删除一个结点，按住Alt移动滚轮控制大小，按住空格键可以上下移动图片，属性中的频率表示加点的频率，对比度是自动识别的色差的大小，按住Alt键可以变为多边形套索工具

4）把一个内容移到另一个内容中：继续打开一个项目，选择移动工具，选择要移动的内容，按住不松，把鼠标指向另一个项目，放入另一个图片中

注：注意图层的关系，边栏中上方的图层在组成时也是在上面的，因此，如果要修改一个图层的背景色，新建的背景色的图层应该在要被加背景的图层的下方

# 5.快速选择和魔棒工具

均用于选择

Alt+鼠标右键=快速选择工具的大小调节

会自动根据颜色识别，按住Alt键再点击可以减选区，细致的地方可以放大进行修改（按住Alt键再滚动鼠标滚轮）（Alt+del=填充前景色）

拆分图层：用矩形选择工具选择一部分后按ctrl+shift+j=拆分图层，ctrl+e向下合并图层

属性中对所有图层取样：如果不选，则识别只是当前的图层

选择并遮住：处理边缘，打开后有属性栏，进行设置，视图处选黑白底可以帮助看边缘，左边框的快速画笔选择工具选择后在边缘涂抹 ，可以自动识别色差，然后自动微调，画笔工具和快速选择工具以及套索工具可以重新选择一些区域进来，净化后没有改变需要先用调整边缘画笔工具把边缘处理淡，先调节好边缘载抠图出来

魔棒工具：通过色彩进行识别，鼠标点击的是什么颜色，它会把与之相似的颜色的区域选择起来，容差是会被选择的内容的大小区域，也可以用加选区，再点一个地方会将新的选区加到之前的选区中，选区可以反选（点击右键->反选，选中与被选中的部分剩余的部分）

# 6.裁剪和切片（标尺使用）

用来裁剪图片（恢复原图：ctrl+z,退出当前的选择的工具可以用esc）

1）裁剪指定大小的图片：属性栏中选长宽像素，来设置固定的尺寸（注：在解除图层的锁定后，勾选属性栏中的自动填充，则在裁剪后按enter后透明区域会根据图片的四周自动填充颜色）

2）拉直图片：选中拉直后，拉一条直线，即图片以这条直线为平面，以这条直线为平面进行旋转（按住ctrl键）

3）设置叠加选项：用来辅助，比如网格线，五等分，三等分等。

3)删除裁剪的像素：选好裁剪区域后用鼠标双击确定后裁剪图片，如果选择了这个选项后则被裁掉的部分就被彻底删除了（按住Alt键，向外拉宽，四个边同时拉宽）

4）透视裁剪工具：可以先框选后选择四角上的一个点进行移动，形成其他多边形状，此时属性栏中的长宽分辨率是裁剪后显示的画布的大小，图片会被压扁在此大小的画布中

5）切片工具：

+ 一般用标尺来做裁剪，视图->标尺，鼠标按住刻度部分拖动，可以上下左右的选择区域，删除一条刻度线可以用移动工具选择一条线后拉会刻度尺部分。
+ 保存各部分的切片：选择切片工具->基于参考线的切片，文件->导出->储存为web所用格式->图片格式选择jpge->存储->保存格式可以选HTML+图像（会同时保存切下的各部分，也会保存一个网页，网页中是原图）
+ 切片的合并：ctrl点击切片---切片选择，ctrl+shift---多选切片，点击右键后可以选择合并，删除切片等，也可以选择--切片选择工具->按住shift键，选择要合并的几个切片
+ 在切片中添加链接：选择切片，右键->url(添加一个链接),然后重复上面保存切片的操作后打开网页，点击切片区域后会打开刚刚设置的那个链接
+ 清除切片：视图->清除切片
+ 显示和隐藏参考线：ctrl+h
+ 新建在某个位置的参考线：视图->新建参考线->水平or垂直->位置（设置像素，可以选择矩形工具框选看某块位置像素的大小）

# 7.图框工具（剪贴蒙版）

+ 剪贴蒙版

便于做贴图（先创建一个区域）

新建画布->新建图层->矩形选框工具->框选区域，填充背景色->文件->置入嵌入对象->选择图片->图片图层右键->选择剪贴蒙版   （效果就相当于将图片放入了之前框选的区域（此区域可以任意，图形文字都可）中，当图片依旧可以移动等等）（相当于把照片放入一个框中），再右键可以选择释放剪贴蒙版（或者按住Alt键不送在两个图层中间可以选择和释放剪贴蒙版）

+ 图框工具：选中工具后->画一个框，选择框这个整体->双击使蓝色边框线移到外层->使用移动工具，就可以产生和剪贴蒙版一样的效果，但不同的是，可以把图片拖出来后把图框图层删掉，再选择照片可以移动回来（alt+滚轮--调节大小，ctrl+0--最适屏幕）
  + 属性设置：属性栏中可以画圆和框

# 8.吸管工具组（颜色取样）

用途：得到当前点击的位置的1像素的颜色，可以打开前景色看当前的颜色



1）吸管工具：

属性栏：

+ 取样大小：3x3平均就是当前的像素的点向外扩展3个像素去得到一个平均值，后面以此类推
+ 样本：去吸取的是所选的图层的颜色，其中所有无调整图层就是吸取的不是看到的调整过的图层，是原本的颜色（调整颜色可以选图层面板中的调整图层的选项，其中可以调节色相饱和度等）
+ 显示取样环：可以更清楚的看到当前吸取的颜色

2）颜色取样器工具：

+ 用途：点击后会出现信息面板，显示当前颜色的rgb颜色，可以在前景色中输入这个颜色的代码，就可以找到对应的颜色
+ 删除取样点：按住Alt键点击要删除的点

3）标尺工具：测量长度或角度这些的，x,y表示坐标，L表示长度

选择单位：编辑->首选项->单位与标尺

设置测量比例：在属性栏中有一个选项是使用测量比例，图像->分析->设置测量比例->自定义（像素长度：图像中为多少像素，逻辑长度：实际生活中的长度，再设置逻辑单位），勾选后显示的就是换算后的实际长度

4）注释工具：点击后有一个小本子出现，可以写上修改的注释

5）计数工具：点击一下后就出现依次的数字，来计数

# 9.污点修复工具组

1）污点修复画笔工具：

先使用工具之前，先新建一个空白图层，选中属性栏中从复合数据中取样仿制数据，然后再使用工具，关闭空白图层后可以看见原图

调节当前的使用的圆和污点差不多大小，按住圆在污点上移动可清除污点，原理是取当前选的位置的颜色和周围的颜色取平均值，然后覆盖

+ 属性栏：

           1. 画笔选项：大小（用于框选的圆的大小，可以按住Alt点击右键再左右拖动鼠标），硬度......
           2. 模式：正常（进行计算，然后融合），替换（直接用周围的颜色进行替换）
           3. 类型：内容识别（识别周围颜色），通过纹理修复（不是人物的纹理，可以创建纹理），近似匹配（直接照搬周围的）

  2）修复画笔工具：

  先按住Alt键取颜色的样（吸周围的颜色），再进行涂抹，连续涂抹可以擦出一个和所选取的图案相同的图案

  + 属性栏： 
    1. 样本：选所有图层
    2. 对齐：不选中可以重新进行相同内容绘制，选中会根据原图然后擦除连续的效果

3）修补工具：类似套索工具，用鼠标画出一个选区，然后拖拽到其他区域可以用其他地方填补，可以保留图像的纹理

4）内容感知和移动工具：框选区域后移动此区域在图像的位置(ctrl+D:取消选择)

结构：保留绘制的结构

颜色：颜色匹配度越高，越能融入周围环境

模式：扩展（移动图像后原图还在），移动（移动后原图不在了）

允许旋转和缩放选区：选择移动后可以改变大小等

# 10.画笔工具

画的颜色是前景色，硬度越大，羽化效果越小，图层和该属性栏中都可以设置不透明度

设置绘图的对称选项：可以画各种方位的对称

# 11.仿制图章工具

用来进行图像中内容的复制

使用：进行取样（按住Alt键，再点击要仿制的内容）->找到目标位置，拖动鼠标去涂抹，就可以把内容复制出来（从取样点开始，进行内容的复制）

在低的不透明度下，反复的进行涂抹，可以加深图像

流量：当前的笔的出水量

样本：当前图层（可以把复制的内容到新的图层上，从而对复制的内容进行操作）

仿制源：可以记录多个点的取样

可以用来去污点：新建图层（避免影响到原图）->选择图章工具->吸取一个点->在污点上涂抹

图案图章工具：可以在图片中间添加一些图案

# 12、历史记录画笔组

和历史记录面板相似，返回上一步操作，不同的是这个工具可以选择进行涂抹，对某一部分进行撤销，进行局部的恢复调节，恢复到的是最原始的状态，如果要设置一个恢复的标准，就在历史记录面板中选择一个状态，然后点击右下角的小相机，然后选择这个状态（点击小方框）

历史记录画笔工具：可以造成类似水彩的效果
